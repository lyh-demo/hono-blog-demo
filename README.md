# hono-blog-demo

CRUD web API for Blog


## Endpoints

- `GET /posts`
- `POST /posts`
- `GET /posts/:id`
- `PUT /posts/:id`
- `DELETE /posts/:id`

## Usage

Install dependencies:

```
pnpm install
```

Run a development server:
```
pnpm run dev
```

Publish:
```
pnpm run deploy
```
